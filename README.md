📊 Sales Data Analysis Using SQL Server
## Overview

This project analyzes sales transaction data using Microsoft SQL Server. The goal is to clean the data, perform business analysis, and generate insights to support decision-making.

## Tools Used
Microsoft SQL Server
SQL
CSV Dataset

## Data Cleaning
Removed duplicate records
Handled missing values
Corrected column names
Standardized gender and payment mode values

## Analysis Performed
1.Top Selling Products
2.Most Cancelled Products
3.Peak Purchase Time Analysis
4.Highest Spending Customers
5.Revenue by Product Category
6.Return & Cancellation Rate Analysis
7.Preferred Payment Methods
8.Age Group Purchasing Behavior
9.Monthly Sales Trends
10.Gender-Based Product Preferences

## Key Insights
Identified best-selling products
Found peak sales hours
Determined top customers by revenue
Analyzed category-wise revenue performance
Evaluated customer purchasing behavior
